1.Java语言中运算符的分类

算数运算符：+，—，，/，++，—，%；
关系运算符：>，<，>=，<=，==（xx是否等于xx），！=；
布尔逻辑运算符：！，&，|，^，&&，||；
位运算符：&，|，^，~，>>，<<，>>>；
赋值运算符：=；
扩展赋值运算符:+=，-=，=，/=
*字符串连接运算符：+

2.几种运算符的实例

int i = 3 / 2 计算出值为1
double i = 3/2.0 计算出值为1.5 表达式的类型取决于可表示最大范围的数据类型

i++ 与 ++i 的区别
i++ 表示 i先参与后面的运算，再加1；
++i 表示 i先加1，然后再参与后面的运算；

boolean 类型的表达式只有两种结果：True 和 False；

练习1：

public class Test1{
    public static void main(String args[]){
    int i = 5;
    int j = 7;
    boolean e = i>j;
    System.out.println(e);
    }
}

练习2：

public class Test2{
    public static void main(String args[]){
    int i = 3;
    boolean e = i > 5 && i++ > 4;
    System.out.println(e);
    System.out.println(i);
    }
}

练习1返回的值为false；练习2中返回的e为false，i为3，因为&&为“短路与”运算符，只需要判断出i>5即可得出e为false，而 i++ > 4这一步也无需再运算，因此i的值返回为 3。

总结：逻辑运算符在编写代码中比较常用，要牢记它的含义，使用方法和规则以及使用的场景。另在编写代码时要养成良好的习惯：
1.主函数要特别熟练地写出来；
2.每一行表达式后应该注意不要遗漏分号（英文版的”;”）；
3.还要注意表达式中的空格。
