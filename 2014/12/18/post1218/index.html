分支语句

分支语句类型共分为两种：if…else语句与switch语句；
（1）if…else语句：
第一种

if(布尔类型表达式){
    语句一；
    语句二；
    ......
}

第二种

if(布尔类型表达式){
    语句一；
    语句二；
    ......
}
else(布尔类型表达式){
    语句一；
    语句二；
    ......
}

第三种：

if(布尔类型表达式){
    语句一；
    语句二；
    ......
}
else if(布尔类型表达式){
    语句一；
    语句二；
    ......
}
else(布尔类型表达式){
    语句一；
    语句二；
    ......
}

（2）switch语句

switch(表达式：char、byte、int、短整型){
    case 常量1:
            语句一；
            breake；
    case 常量2:
            语句二；
            break；
    ......
    [default:
        默认语句；
        break;]
}

注：break 的含义是不再往下执行语句命令。
