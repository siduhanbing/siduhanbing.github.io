Java基础学习（继承、在子类中调用父类的构造函数）

By 黄静函 胡洋

1月 7 2015 更新日期:1月 7 2015

  1. 1. 一.Java语法中的继承
    1. 1.0.1. 1.学习感悟。在Java语法中，继承是一个重要的概念和方法。继承也很实用，为程序员提高编写代码效率提供了很好的方法。假设在不使用继承的情况下，一个程序可能包含千千万万个类，这些类之间可能存在相同的代码，而如果需要对这些代码进行修改，就需要对各个类进行修改，极大阻碍了代码编写效率，不符合程序员一直坚持的“消灭重复代码”的理念，消灭重复代码使程序顺畅执行才是程序员的基本使命。
    2. 1.0.2. 2.为避免上述问题的产生，继承会有效提升程序员的工作效率，同时也会使代码简洁、漂亮。简单来说，使用继承必然会出现父类和子类。实现的方法就是利用“extend”命令来使子类来继承父类中的成员变量和成员函数（注：Java语法中 函数=方法）。如果父类中包含构造函数，这时子类则不能再继承父类的构造函数，但子类可以而且必须调用父类的构造函数。如果父类构造函数默认没有参数，这时子类无需任何指令则可以实现调用；如果父类构造函数中含有参数，这时子类则需要通过“super”来实现调用。

    二.实例展示

    1.创建一个名为Family的父类并赋予成员函数变量与构造函数变量：

    class Family{
    String name;
    String hobby;
    int age;
    //以上为成员变量

    Family(){
        System.out.println("无参数的Family构造函数");
    }
    //以上是为Family创造的没有参数的构造函数

    Family(String name,int age){
        this name = name;
        this age =age;
    }

    void work(){
        System.out.println("工作");
    }


}

    2.创建一个名为Father并集成Family成员函数的子类

    class Father extends Family{
    Father(){
        System.out.println("无参数的Father构造函数");
    }
    String workplace;
    Father(String name,int age,String workplace){
        super(name,age);
        String workplace = workplace;
    }

}
